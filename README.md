# AI Fake News Detector PRO

A Machine Learning Web App that detects Fake News.

## Features

- Detect Fake / Real News
- Check News from URL
- Machine Learning Prediction
- Visualization

## Tech Stack

Python  
Streamlit  
Scikit-learn  
TF-IDF NLP

## Run

Install packages

pip install -r requirements.txt

Train model

python model/train_model.py

Run app

streamlit run app.py
